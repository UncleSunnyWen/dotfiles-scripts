## dotfiles-scripts

* 个人开发环境配置文件与实用脚本集合，涵盖窗口管理器、终端、状态栏等定制化配置

### 仓库内容

#### 系统 / 应用配置文件

* .config/：Linux桌面环境配置
    * i3/：i3 窗口管理器配置（含config主配置、i3exit退出脚本）
    * mpd/：音乐播放器守护进程（MPD）配置
    * polybar/：Polybar 状态栏配置（含config主配置、launch.sh启动脚本）
    * ranger/：Ranger 文件管理器配置（命令扩展、界面样式、预览脚本）
    * rofi/：Rofi 启动器配置（主题与行为设置）
    * user-dirs.dirs：用户目录（桌面 / 文档等）自定义配置

#### 终端与工具配置

* .zshrc：Oh My Zsh配置
* .oh-my-zsh/custom/plugins/：Oh My Zsh 自定义插件（adb、eza 等工具扩展）
* .tmux.conf：Tmux 终端复用工具配置（快捷键、面板布局、会话管理）

#### 实用脚本与工具

* monkey.sh：Android Monkey 测试 / 调试辅助脚本
* Android_Log.bat：Windows 下 Android 日志抓取脚本
* Jesus: Python script for hospital appointment registration
